The dataset used in this project offers football match data from more than 30 leagues worldwide. The data spans from the 2000/01 season to the most recent results from the 2024/25 season. 
https://www.kaggle.com/datasets/adamgbor/club-football-match-data-2000-2025

Our objective is discover between the top 10 European leagues, based in Opta Power Rankings in 02/04/2026, which leagues have:
https://theanalyst.com/articles/strongest-football-leagues-in-the-world-opta-power-rankings

PRE REQUISITOS 

1. Ter uma conta no Google Cloud, com uma billing account e um projeto habilitados.
2. Ter o Terraform instalado
https://developer.hashicorp.com/terraform/install
3. Ter uma conta no Kaggle para obter nome de usuário e token.
Settings >> API Tokens >> Generate New Token


The commands used below are intended for the Linux environment.

Com base nos pré-requisitos, no seu terminal crie as variáveis:

#Google Cloud Project ID
export GCP_PROJECT_ID=your_gcp_project_id

#Kaggle Username
export KAGGLE_USERNAME=your_username

#Kaggle API Token
export KAGGLE_API_TOKEN=your_api_key

PASSOS

3. Clonar o repositorio para a sua estação

git clone <URL>

4. Acesse o  arquivo terraform.tfvars.example em cd terraform/ e informe os valores das variáveis. Todas elas se referem ao Google Cloud como por exemplo: project_id
  4.1. informe os valores das variáveis 
  4.2. Renomeie o arquivo para terraform.tfvars
  4.3. Copie o valor de 'composer_env_name' e salve em uma variável no terminal: 
    export COMPOSER_ENV_NAME=<nome>

5. Autentique-se no GCP 

gcloud auth application-default login

  5.1 Valide se está no projeto correto:

  gcloud config get-value project

  Caso não esteja...

  gcloud config set project <GCP_PROJECT_ID>

Depois execute:

gcloud auth application-default set-quota-project seu-project-id

6. Ainda no diretório cd terraform/ execute os comandos:

terraform init
terraform plan
terraform apply

Aguarde até o ambiente ser provisionado (pegue um café).
