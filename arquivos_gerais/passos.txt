The dataset used in this project offers football match data from more than 30 leagues worldwide. The data spans from the 2000/01 season to the most recent results from the 2024/25 season. 
https://www.kaggle.com/datasets/adamgbor/club-football-match-data-2000-2025

Our objective is discover between the top 10 European leagues, based in Opta Power Rankings in 02/04/2026, which leagues have:
https://theanalyst.com/articles/strongest-football-leagues-in-the-world-opta-power-rankings

PRE REQUISITOS 

1. Ter uma conta no Google Cloud, com uma billing account e um projeto habilitados.

2. Google SDK instalado

3. Ter o Terraform instalado
https://developer.hashicorp.com/terraform/install

4. Ter uma conta no Kaggle para obter nome de usuário e token.
Settings >> API Tokens >> Generate New Token

4. Ter o Docker instalado 

----------------

The commands used below are intended for the Linux environment.



git clone <repo>
cd european-football-analytics-2000-25
cp .env.example .env   # preencher as variáveis

cd terraform
cp terraform.tfvars.example terraform.tfvars # preencher as variáveis

5. Autentique-se no GCP 
gcloud auth login

  5.1 Se você tem mais de um projeto ativo no GCP, valide se está no projeto correto:

  gcloud config get-value project

  Caso não esteja...

  gcloud config set project <GCP_PROJECT_ID>

Ainda dentro da pasta de Terraform

terraform init
terraform plan
terraform apply

Aguarde até o ambiente ser provisionado

Volte para raiz do projeto:

docker compose build   # primeira vez — usa o cache a partir daqui
docker compose up airflow-init   # migra o banco + cria admin
docker compose up -d        # Build the custom image and start the containers in detached mode
bash setup.sh               # seta Airflow Variables


Acessa localhost:8080

airflow
